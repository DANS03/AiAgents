# 🤖 AI Agent Development & Orchestration

Welcome to my coursework repository! This repo contains all my projects, code, and notes from my class on building and managing AI agents. 

The goal of this repository is to explore the cutting edge of autonomous systems, multi-agent collaboration, and AI workflow automation using a mix of Python, TypeScript, and industry-standard frameworks.

## 📚 Topics & Technologies Covered

This repository is organized around the following core frameworks and concepts:

* **OpenAI Agents SDK:** Building native, intelligent agents using the official OpenAI toolsets and APIs.
* **LangGraph:** Creating stateful, multi-actor applications and cyclical graphs for complex agent reasoning.
* **N8N:** Designing node-based AI workflows and automating integrations between diverse services.
* **CrewAI:** Orchestrating role-playing autonomous AI agents to collaborate seamlessly on complex tasks.
* **AutoGen:** Utilizing Microsoft's framework for developing multi-agent conversational systems.
* **ChatGPT Agents & CoPilot:** Exploring custom GPTs, agentic assistants, and AI-assisted development workflows.
* **MCP (Model Context Protocol):** Implementing standardized protocols to connect AI models with external data sources, APIs, and tools.
